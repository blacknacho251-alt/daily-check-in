# Daily Check-in

GitHub Actions 驱动的上下班打卡项目。

### 如何使用
1. 点击仓库顶部的 **Actions** 选项卡。
2. 在左侧选择 **Work Clock In/Out**。
3. 点击 **Run workflow** 下拉菜单，选择打卡类型（Clock In 或 Clock Out）。
4. 点击 **Run workflow** 按钮完成打卡。
